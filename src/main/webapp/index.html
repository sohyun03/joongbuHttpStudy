http 수업을 위해 다이나믹 웹 프로젝트 생성

  • a 태그로 리소스 참조
  • 리소스: 참조할 수 있는 자료
  • 웹 상의 리소스: 인터넷에서 참조할 수 있는 자료 (브라우저 url 주소로 참조할 수 있는 것, HTTP(HyperTextTransferProtocal) 통신으로 참조할 수 있는 것)
  • 정적 리소스: 참조할 때 마다 변화가 없는 자료 (html, css, js, img, mp3, mp4...) => frontend: 화면을 꾸미거나 브라우저에서 동작
  • 동적 리소스: 참조할 때 마다 변하는 자료 (java, phython, php, asp, node....) => backend: data를 처리하거나 서버에서 실행
  • 동적 리소스(servlet) 참조하기
